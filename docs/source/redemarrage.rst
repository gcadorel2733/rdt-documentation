Redémarrage de Home Assistant
--------
Pour redémarrer Home Assistant :
   1. Cliquer sur **Paramètres**
            .. image:: images/parametres.png
               :width: 150
               :align: center
   2. Choisir **Système**
         .. image:: images/systeme.png
            :width: 500
            :align: center
   3. Cliquer en haut à droite sur le bouton.
            .. image:: images/reboot.png
               :width: 50
               :align: center
