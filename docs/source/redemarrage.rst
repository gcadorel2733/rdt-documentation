Redémarrage de Home Assistant
-----------

Pour redémarrer Home Assistant :
   - Cliquer sur **Paramètres**
   - Choisir **Système**
   - Cliquer en haut à droite sur le bouton.

.. code-block:: console

   (.venv) $ pip install lumache
